# Shoes for Kicks
Intro webdevelopment project 2

# Opgave tekst
I denne opgave får I udleveret HTML filer, billedfiler, og 3 billeder af undersider til hjemmesiden "Sko og Tricks". Opgaven går ud på at få de tre sider til at ligne de udleverede billeder. 

Der er i sagens natur ikke mål på billederne, så du skal bruge lidt logisk sans for at regne ud, hvordan billedernes visuelle udtryk omsættes til et færdigt website. 

I de udleverede html filer mamgler der nogle ting, for at siderne kan komme til at ligne billederne.

- Du må gerne  tilføje manglede attributter eller billeder samt tilføje klassenavne til de udleverde html filer.

- Du må gerne oprette én CSS fil. Den skal kunne bruges til alle sider. I filen  den CSS der mangler, for at det kommer til at se ud som det gør i de  3 billeder.

- Du må ikke ændre på HTML filernes struktur.

Index.html har ikke noget indhold. Lav den når du er færdig med de anre sider. Her er der frit slag i forhold til struktur.

Sammen med opgaven skal du aflverer et tekst-dokument som en mark-down fil. I denne fil skal du beskrive, hvilke ændringer du ville lave HTML strutruren, så den blev mere optimal, eller måske mere semantisk korrekt.
